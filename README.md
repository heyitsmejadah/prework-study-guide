
# <Prework Study Guide Webpage>

## Description
I needed to create this study guide because it would help me organize my learning materials in an online environment. It helps keep everything in a place where I know it will be an allows me to test JS, CSS, HTML, and GIT in a safe place where I cannot mess anything up.

## Usage
Each section contains notes about a certain topic, like HTML, CSS, JavaScript, Git. If we open the DevTools console, we can see the JavaScript running. It will list the topics we learned from prework and choose one topic for us to study first.
